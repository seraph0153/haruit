아래 링크에서 바로 사용할 수 있습니다:

### 홈 화면에 추가 (앱처럼 사용)

**아이폰**:
1. Safari로 웹앱 접속
2. 하단 공유 버튼 (📤) 탭
3. "홈 화면에 추가" 선택
4. "추가" 탭

**안드로이드**:
1. Chrome으로 웹앱 접속
2. 우측 상단 메뉴 (⋮) 탭

4. "추가" 탭

## 🌟 주요 기능

### 사용자 (시니어)
- 거동 상태 선택 → 환경 스캔 → AI 미션 받기
- 스몰토크 기능 (주변 사람과 대화 유도)
- 활동 기록 및 달력 뷰

### 관리자 (기관)
- 사용자 참여 현황 대시보드
- 개별 사용자 통계 확인
- 데이터 내보내기 (CSV)

## 🔒 프라이버시
- 사진/영상 저장 없음
- 모든 데이터는 사용자 기기에만 저장 (LocalStorage)

## 🛠️ 기술 스택
- HTML5 / CSS3 / JavaScript
- LocalStorage (서버 불필요)
- 반응형 디자인 (모바일 우선)
